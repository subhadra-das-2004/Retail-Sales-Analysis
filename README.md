# Retail Sales Analysis

## Project Overview

This project performs exploratory data analysis (EDA) on a real-world retail sales dataset to identify key business insights and revenue drivers.

## Dataset

* Sales Transactions
* Product Hierarchy
* Store Information

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Analysis Performed

* Data Cleaning
* Missing Value Handling
* Statistical Summary
* Top Products Analysis
* Top Stores Analysis
* Revenue Distribution
* Correlation Heatmap
* Product Cluster Analysis
* City-wise Revenue Analysis

## Key Findings

* Product P0129 generated the highest revenue.
* Store S0097 was the top-performing store.
* Cluster_0 contributed the highest revenue.
* City C014 generated the largest share of revenue.
* Sales and revenue showed strong positive correlation.

## Business Impact

The analysis helps identify profitable products, stores, and cities, enabling better inventory management and business strategy decisions.
